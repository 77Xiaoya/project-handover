# Documentation Index

This folder contains the public-facing handover material for the project.

## What To Read First

If you are new to the repository, the recommended reading order is:

1. `../README.md`
2. `script-purposes.md`
3. `raspberry-pi-section-bilingual.md`
4. the wiring notes that match the hardware you want to inspect

## Main Documents

- `raspberry-pi-section-bilingual.md`: Raspberry Pi-focused bilingual write-up
- `script-purposes.md`: script and message reference

## Wiring Notes

- `joystick-wiring.md`: joystick module wiring and signal notes
- `slider-wiring.md`: slider module wiring notes
- `encoder-wiring.md`: encoder power, click, and GPIO notes

## What Is Not Public Here

- Private figure assets were archived outside the public repository.
- Readability-focused draft material was archived outside the public repository.
- Heavier report source files were archived outside the public repository.

## Notes

- This public `docs/` folder is intentionally small and focused on handover reading.
