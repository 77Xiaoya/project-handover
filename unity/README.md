# Unity Script Mirror

This folder mirrors the most important Unity-side scripts in a GitHub-friendly location.

- `PiSystemBridge.cs`: Raspberry Pi UDP receiver and interaction bridge
- `WaterSystemManager.cs`: UI state, data filtering, and chart controller
- `mapswitch.cs`: map layer switching logic
- `IntersectionRiverDetector.cs`: river intersection helper
- `waterQualityP.cs`: water-quality presentation helper
- `maplayermanager.cs`: map layer support logic

These files are copied from the Unity project so reviewers can inspect them directly on GitHub.

## Recommended Reading Order

1. `PiSystemBridge.cs`
2. `WaterSystemManager.cs`
3. `mapswitch.cs`

The remaining files are supporting scripts used by the Unity scene.
